## 🌐 Обновление только UI (web-only OTA)

Начиная с v2.8.0 статика веб-интерфейса хранится в отдельном разделе `spiffs_web` и может обновляться отдельно от ядра прошивки.

### Загрузка UI-образа

1. Соберите `web_spiffs.bin` командой:
   ```bash
   pio run -e esp32dev -t buildfs  # файл .pio/build/esp32dev/web_spiffs.bin
   ```
2. Отправьте бинарник на устройство:
   ```bash
   curl -X POST http://<IP>/api/ota/web_only --data-binary @web_spiffs.bin
   ```
3. Устройство примонтирует новый образ без перезагрузки.

> Эндпоинт возвращает `{ok:true}`. Пока это PoC – проверка SHA-256 и откат будут добавлены позже.
